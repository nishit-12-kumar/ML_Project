About the ML Math Predictor

ML Math Predictor is a machine learning-powered web application that predicts a studentâ€™s mathematics score based on various inputs including gender, race or ethnicity, parental education level, lunch type, and scores in reading and writing.

The system evaluates several regression models like Random Forest, Decision Tree, Gradient Boosting, Linear Regression, K-Neighbors Classifier, XGBRegressor, CatBoosting Regressor, AdaBoost Regressor . The model with the best performance on the dataset is used to ensure high prediction accuracy. The final model used for prediction achieves an accuracy of 89%.

The form you see on the homepage collects all the required inputs and sends them to the backend where the trained model makes a real-time prediction.

The training data used to build this model is sourced from real-world student academic performance datasets.
